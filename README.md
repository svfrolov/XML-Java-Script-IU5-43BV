# XML и JavaScript

Репозиторий для лабораторных работ по XML и JavaScript (ИУ5-43БВ)

## Структура репозитория

- **lab1** - Калькулятор на JavaScript
- **lab2** - [Описание второй лабораторной работы]
- **lab3** - [Описание третьей лабораторной работы]
- **lab4** - [Описание четвёртой лабораторной работы]
- **lab5** - [Описание пятой лабораторной работы]
- **lab6** - [Описание шестой лабораторной работы]
- **lab7** - [Описание седьмой лабораторной работы]
- **lab8** - [Описание восьмой лабораторной работы]

## Автоматический деплой

Репозиторий настроен на автоматический деплой на сервер при каждом push в ветку main.
Для каждой лабораторной работы создается отдельная директория на сервере.
