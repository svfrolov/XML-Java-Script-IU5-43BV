# XML и JavaScript

Репозиторий для лабораторных работ по XML и JavaScript (ИУ5-43БВ)

## Структура репозитория

- **lab1** - Калькулятор на JavaScript
- **lab2** - [Файлы второй лабораторной работы]
- **lab3** - [Файлы третьей лабораторной работы]
- **lab4** - [Файлы четвёртой лабораторной работы]
- **lab5** - [Файлы пятой лабораторной работы]
- **lab6** - [Файлы шестой лабораторной работы]
- **lab7** - [Файлы седьмой лабораторной работы]
- **lab8** - [Файлы восьмой лабораторной работы]

## Автоматический деплой

Репозиторий настроен на автоматический деплой на сервер при каждом push в ветку main.
Для каждой лабораторной работы создается отдельная директория на сервере.
